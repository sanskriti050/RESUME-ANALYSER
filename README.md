## 🚀 Overview

ResumeLens AI is an intelligent resume analysis platform that helps students and job seekers improve their resumes using AI.

The application analyzes resumes, calculates ATS compatibility, identifies strengths and weaknesses, generates personalized career recommendations, creates professional cover letters, suggests interview questions, builds learning roadmaps, and even allows users to chat with an AI assistant about their resume.

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/ResumeLens-AI.git

cd ResumeLens-AI
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Create .env

```env
GROQ_API_KEY=your_api_key_here
```

---

## Run

```bash
streamlit run app.py
```
